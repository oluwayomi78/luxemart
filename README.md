## Overview
This is a modern **luxemart website** built to provide users with a seamless online shopping experience.  
Users can browse products, add them to the cart, make purchases, and leave reviews. The platform is designed with a clean, responsive UI for desktop, tablet, and mobile.

## Features
- User authentication (signup/login)
- Browse products by category and brand
- Add products to cart and wishlist
- Checkout and payment integration (placeholder or real)
- Product reviews and ratings
- Admin dashboard for managing products, orders, and users
- Responsive and mobile-friendly design

## Technologies Used
- **Frontend:** React, Tailwind CSS,
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Other Tools:** Axios, JWT (authentication), Cloudinary (image hosting), Redux
